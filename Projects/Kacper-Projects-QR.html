About QR Generator

How it works: The application features a text field where users can input new tasks. When a user enters a new task and hits the Enter key, it gets added to the list. Tasks can be marked as completed or unchecked by clicking on them. Each task has a trash icon for deletion upon clicking. The count of pending tasks to be completed is displayed for user reference.

Technologies used in a nutshell: HTML, CSS, JavaScript: These are web programming languages used for content creation, styling, and website interactivity. HTML: Structure: Elements such as text fields < textarea >, lists < ul >, buttons < button >, and other UI components are structurally defined using HTML tags. CSS: Styling: Within the style section, CSS styles define the appearance and layout of elements on the page. Styling includes aspects like background color background-color, margins margin, fonts font-family, font-size, and scrollbar appearance scrollbar. JavaScript: User Interactions: The JavaScript code within the script > section handles user interactions with the application. It manages adding new tasks (addEventListener), marking tasks as completed or undoing that action (onclick, classList.toggle), deleting tasks (remove), and updating displayed data on the page. These technologies are used across different facets of the webpage, ensuring comprehensive functionality for managing a task list.
